# Hybrid Object Detection Project with SAIOD Dataset 🛰️🤖

This project implements a hybrid machine learning and computer vision structure designed for object detection on aerial imagery. It utilizes the **SAIOD (Satellite Aerial Images for Object Detection)** dataset to train and evaluate models for identifying specific objects from satellite and aerial perspectives.

### 🛠️ Built With / Kullanılan Teknolojiler
* **Language:** Python
* **Core Field:** Computer Vision (CV), Deep Learning & Remote Sensing
* **Dataset:** SAIOD (Satellite Aerial Images for Object Detection)

---

### 🇹🇷 Proje Hakkında (Turkish)
Bu proje, hava ve uydu görüntüleri üzerinde nesne tespiti (object detection) gerçekleştirmek amacıyla **hibrit bir yapıda** geliştirilmiştir. Çalışmada **SAIOD** veri seti kullanılmış olup, yüksek çözünürlüklü uzaktan algılama görüntülerindeki nesnelerin tespiti ve sınıflandırılması için bilgisayarlı görü ve derin öğrenme yaklaşımları optimize edilmiştir.
